# Character Counter

A clean and modern Character Counter app built with **HTML**, **CSS**, and **JavaScript**.

It updates in real time while the user types and displays useful text statistics in a simple and professional UI.

---

## ✨ Features

- Live character count
- Live word count
- Sentence count
- Space count
- Clean dark modern UI
- Responsive design for desktop and mobile

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

---

## 📚 What I Practiced

This project helped me practice:

- DOM selection
- Event listeners
- Working with user input
- Updating the UI dynamically
- String methods like `length`, `trim()`, and `split()`
- Basic regex with `match()`

---

## 🚀 How It Works

The user types into the textarea, and the app updates the statistics instantly.

It counts:

- **Characters** using `text.length`
- **Words** using `trim()` and `split()`
- **Sentences** using regex with `match()`
- **Spaces** using regex with `match()`
